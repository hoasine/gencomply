# GenComply Frontend

Next.js dApp for **GenComply** — on-chain policy registry and compliance-violation bounties on GenLayer Studionet.

**Author:** Hoa Tran Rom  
- X: [https://x.com/HoaTranRom](https://x.com/HoaTranRom)  
- GitHub: [https://github.com/hoasine](https://github.com/hoasine)

## Setup

1. Install dependencies:

```bash
npm install
```

2. Create `.env` file:

```bash
cp .env.example .env
```

3. Set `NEXT_PUBLIC_CONTRACT_ADDRESS` (from GenLayer Studio deploy).

## Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

Social links also appear in the app header and footer.
